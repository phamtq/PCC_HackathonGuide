# DOFBOT-Pi Guide

## Hardware Setup

Ensure power is plugged into the bottom black board, connected to the wall / power adapter, switch power on, and lights should turn on near OLED screen.

On raspberry pie, make sure the microSD card in inserted correctly (will feel like it's upside down), 

Plug in screen, keyboard, and mouse.

Connect to WiFi (Dofbot). Password: 12345678

SSH pi@192.168.1.11, password: yahboom

WARNING: If you hear rapid clicking noise: Immediately switch off the entire station. Rotation trying to exceed limits, possible breakage.
